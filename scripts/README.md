# Scripts

Bootstrapping scripts to run Sovereign.

- Get an OpenTopography API key at https://portal.opentopography.org/.
- Set `OPENTOPO_API_KEY` in your environment.

|Script       |Purpose                                                         |
|-------------------------------------------------|----------------------------------------------------------------|
|[dem_dl.py](./dem_dl.py)           |Download open-source topography `dem` files for Cesium terrain maps.                   |
